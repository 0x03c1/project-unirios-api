# API BCB consultas dos índices

1. Criar o ambiente virtual

```bash
python -m venv .venv
```

2. Ativar o ambiente virtual no Linux

```bash
source .venv/bin/activate
```

3. Ativar o ambiente virtual no Windows

```bash
.venv\Scripts\activate
```

4. Instalar as bibliotecas necessárias para o funcionamento do projeto

```bash
pip install -r requirements.txt
```
